# TareasApp

TareasApp es una aplicación en la cual se realiza creacion, eliminacion, modificacion y vista de Tareas. Utilizando autenticación JWT.

## Requisitos

* Visual Studio
* SQL Server
* .NET 8

## Utilización

1) Clonar repositorio.

```git
git clone https://github.com/DwGonzalez/TareasApp.git
```

2) Abrir proyecto en Visual Studio.
3) Restaurar paquetes de NuGet en la solución. (opcional)
4) Asegurarse que la dirección de SQL este correcta (appsettings.json).
5) En la consola de Administrador de Paquetes introducir:
```
Update-Database
```
6) Ejecutar proyecto.
